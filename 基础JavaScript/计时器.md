# 计时器

`setInterval()` 定时调用, 可以将一个函数, 每隔一段时间执行一次

参数: 1. 回调函数   2. 每次间隔时间 (ms)
返回值: 返回一个Number类型的数据, 作为唯一标识

```js
var time = document.querySelector('#time');
var num = 0;
setInterval(function(){
	time.innerHTML = num++;
},100);
```

### 取消定时器

`clearInterval()` 参数 计时器

```js
var timer = setInterval(function(){
	time.innerHTML = num++;
	if (num == 11) {
		clearInterval(timer);
	}
},100);
```
